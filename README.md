# Analytics Dashboard

Interactive analytics dashboard with charts, stats, and data visualization.

## How to Use

**Step 1:** Open `index.html` in your browser.

**Step 2:** An onboarding tutorial appears on first visit. Click **Open Dashboard** (or Skip).

**Step 3:** View the 4 key metric cards at the top: Visitors, Revenue, Conversions, and Bounce Rate. Each shows the current value and trend percentage.

**Step 4:** Analyze the bar chart on the left showing traffic data. Hover over any bar to see the exact value.

**Step 5:** View the donut chart on the right showing traffic sources (Direct, Organic, Social, Referral, Email).

**Step 6:** Click **Week**, **Month**, or **Year** buttons to switch the data range. All charts and stats update instantly.

**Step 7:** Scroll down to see the Recent Activity table with page views, duration, and status badges.

## Features
- 4 stat cards with trend indicators
- Interactive bar chart (hover tooltips)
- Donut chart for traffic sources
- Time period toggle (Week/Month/Year)
- Activity table with status badges
- Fully responsive design

## Tech
HTML + CSS + JavaScript | No chart library needed
